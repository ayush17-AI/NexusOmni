# NexusOmni – Esports Intelligence Platform

NexusOmni is a futuristic esports dashboard built using Next.js and Tailwind CSS. The platform focuses on the Indian BGMI esports ecosystem and features a cinematic space-themed interface with glassmorphism UI and fullscreen portals.

## Project Overview
NexusOmni provides a comprehensive view of the competitive esports landscape, offering detailed player stats, team insights, and tournament data. The interface is designed to be immersive, reflecting the high-energy world of competitive gaming.

## Features
- **Cinematic UI**: Atmospheric space-themed design with glassmorphism and subtle animations.
- **Esports Portals**: Quick access to dedicated hubs for BGMI, Free Fire, and COD Mobile.
- **Player Standings**: Live-style dashboard for tracking top Tier 1 players.
- **Responsive Layout**: Forced fullscreen design to maintain visual consistency across devices.
- **Dynamic Content**: Data-driven components for players and teams.

## Tech Stack
- **Next.js** (App Router)
- **React**
- **Tailwind CSS**
- **TypeScript**
- **Turbopack**
